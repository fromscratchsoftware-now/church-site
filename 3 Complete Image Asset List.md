### **üì¶ How to Replace Images**

#### **Option 1: Upload Your Own Images**

You can upload images to your project and reference them:  
code  
// Instead of Unsplash URL:  
image: "https://images.unsplash.com/photo-XXX?w=600\&q=80"

// Use your uploaded image:  
image: "/images/sermon-thumbnail.jpg"

Steps:

* Create `public/images/` folder in your project  
* Add your images there  
* Reference as `/images/filename.jpg`

#### **Option 2: Use PocketBase File Storage**

code  
// Upload to PocketBase and get URL:  
const imageUrl \= pb.files.getUrl(record, record.thumbnail)

// Use in your code:  
image: imageUrl

#### **Option 3: Keep Unsplash (Current)**
